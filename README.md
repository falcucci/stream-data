
















# stream-data

Collect the stream data and send it into a callback function.




## Installation

```sh
$ npm i stream-data
```









## Example






```js
"use strict";

const streamData = require("stream-data");

console.log(streamData());
```






## Documentation





### `streamData(a, b)`
Collect the stream data and send it into a callback function.

#### Params
- **Number** `a`: Param descrpition.
- **Number** `b`: Param descrpition.

#### Return
- **Number** Return description.
